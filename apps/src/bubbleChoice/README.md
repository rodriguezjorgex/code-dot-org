# Bubble Choice

This is a React client for a `bubble_choice` level using [Lab2](../lab2/)..

Note that this is only used for levels that use Lab2.  For levels that don't use Lab2, they will get
an older-style level implemented with [a HAML page](https://github.com/code-dot-org/code-dot-org/blob/e33e53386330161b1b725c34f73aca81c9093cba/dashboard/app/views/levels/_bubble_choice.html.haml), an [associated entrypoint](https://github.com/code-dot-org/code-dot-org/blob/e33e53386330161b1b725c34f73aca81c9093cba/apps/src/sites/studio/pages/levels/_bubble_choice.js), and [a React component](https://github.com/code-dot-org/code-dot-org/blob/e33e53386330161b1b725c34f73aca81c9093cba/apps/src/code-studio/components/BubbleChoice.jsx).
